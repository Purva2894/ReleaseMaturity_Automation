# Release Maturity Automation Test Framework

## Overview
This is a **Java + Cucumber + Selenium** automation framework for testing the **Release Maturity** application.

## Prerequisites
- Java 17+ installed
- Maven installed
- ChromeDriver (or WebDriverManager)
- GitHub account
